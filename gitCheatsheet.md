# Git Cheatsheet

## Create a new Repository
echo "# test" >> README.md

	git init

	git add README.md

	git commit -m "first commit"

	git branch -M main

	git remote add origin https://github.com/TmAwotunde/test.git

	git push -u origin main

## Push an existing Repository

	git remote add origin https://github.com/TmAwotunde/test.git

	git branch -M main

	git push -u origin main
